# Coders of Bangalore - Python Data Parsing

## Overview
This project reads raw Instagram profile data from a text file, converts it into structured Python dictionaries, exports the data to JSON, and performs basic analysis.

## Features
- Read data from a text file
- Parse Instagram profile information
- Convert data into JSON format
- Find:
  - Profile with maximum posts
  - Profile with maximum followers
  - Profile following the most people
  - Total unique page categories
